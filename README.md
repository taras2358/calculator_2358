# Simple Calculator

### Description

Simple calculator which operates numbers in range 0..99

### Tech

* Rails
* Webpack (react_on_rails gem)
* Rspec (test coverage)

### Versions

* Ruby version - 2.5.3
* Rails version - 5.2.2

### Starting the application

You could use foreman for starting application and webpack dev server at once

```sh
gem install foreman
foreman start
```
